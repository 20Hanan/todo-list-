# 📝 Vanilla JavaScript Todo List

A simple and interactive **Todo List application** built with **pure JavaScript, HTML, and CSS**.

This project allows users to manage daily tasks, mark them as completed, and delete tasks.

---

## 🚀 Features

* ➕ **Add New Tasks**
  Users can add tasks with a title.

* ✔️ **Mark Tasks as Completed**
  Click to mark tasks as done.

* ❌ **Delete Tasks**
  Remove tasks you no longer need.

* 🔄 **Dynamic UI Updates**
  The list updates instantly when tasks are added, completed, or removed.

* 💾 *(Optional)* Persistent storage using **localStorage** to save tasks between sessions.

---

## 🛠️ Built With

* HTML5
* CSS3
* **JavaScript (Vanilla JS)**

## 🎯 Project Purpose

The goal of this project is to practice **manipulating the DOM using JavaScript**, handling **user interactions**, and building a **dynamic task management interface** without frameworks.

---

## 📚 What I Learned

* Manipulating the **DOM** to add, remove, and update elements dynamically
* Handling **user events** like clicks and input submission
* Managing state in **pure JavaScript**
* Optionally storing and retrieving data using **localStorage**
* Structuring a small project with **HTML, CSS, and JS separation**
